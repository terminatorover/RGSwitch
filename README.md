# RGSwitch
RGSwitch is a "Fluid Switch" inspired by Leonardo Zakour http://www.materialup.com/posts/fluid-switch

To use it simply get the RGSwitch.h and RGSwitch.m files into your project
and #import "RGSwitch.h"
